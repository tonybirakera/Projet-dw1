# Mon premier site web statique

Ce projet représente mon premier site web développé uniquement avec HTML5 & CSS3 dans le cadre de ma formation en programmation analyste.

Objectif du projet
Créer un site web statique afin de comprendre :
- La structure d’une page web (HTML)
- La mise en forme et le design (CSS)
- L’organisation propre du code
- La séparation entre contenu et présentation
  
# Technologies utilisées
- HTML5
- CSS3
- Visual Studio Code

# Ce que j’ai appris
- Structurer une page avec des sections (header, navigation, contenu, footer)
- Utiliser les classes et les sélecteurs CSS
- Mettre en place une mise en page claire et responsive
- Améliorer la lisibilité et l’organisation du code

# Évolution
Ce projet marque le début de mon parcours en développement web.  
Je continue à approfondir mes compétences en programmation et à développer de nouveaux projets.
Ce projet démontre également ma capacité à transformer des concepts théorique en réalisation concrète.
